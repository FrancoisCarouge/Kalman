A user-defined algebraic division solution.

There exists several ways to find X in X = lhs * rhs^-1 for different tradeoffs. The user provides their implementation. Often, matrix inversion is avoided by solving X * rhs = lhs for rhs through a decomposer.

Examples
kf_6x2x0_vehicle_location.cpp, and kf_8x4x0_deep_sort_bounding_box.cpp.
