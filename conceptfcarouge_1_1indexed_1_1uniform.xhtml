template<typename Matrix>
concept fcarouge::indexed::uniform = []() {
bool result{true};
for_constexpr<0, Matrix::rows, 1>([&result](auto i) {
for_constexpr<0, Matrix::columns, 1>([&result, &i](auto j) {
result &= std::is_same_v<element<Matrix, i, j>, element<Matrix, 0, 0>>;
});
});
return result;
}()

Detailed Description

Every element types of the matrix are the same.

Matrices with uniform types are type safe even with the traditional operators.

Note
A matrix may be uniform with different row and column indexes.
