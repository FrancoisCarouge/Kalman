31 // Initialization
32 // Before the first iteration, we must initialize the Kalman filter and
33 // predict the next state (which is the first state). We don't know what the
34 // temperature of the liquid is, and our guess is 10°C.
35 k.x(10.);
36
37 // Our guess is very imprecise, so we set our initialization estimate error σ
38 // to 100. The estimate uncertainty of the initialization is the error
39 // variance σ^2: p0,0 = 100^2 = 10,000. This variance is very high. If we
40 // initialize with a more meaningful value, we will get faster Kalman filter
41 // convergence.
42 k.p(100 * 100.);
43
44 // Prediction
45 // Now, we shall predict the next state based on the initialization values. We
46 // think that we have an accurate model, thus we set the process noise
47 // variance q to 0.0001.
48 k.q(0.0001);
49
50 k.predict();
51
52 assert(10 == k.x() &&
53 "Since our model has constant dynamics, the predicted estimate is "
54 "equal to the current estimate: x^1,0 = 10°C.");
55 assert(10000.0001 - 0.001 < k.p() && k.p() < 10000.0001 + 0.001 &&
56 "The extrapolated estimate uncertainty (variance): p1,0 = p0,0 + q = "
57 "10000 + 0.0001 = 10000.0001.");
58
59 // Measure and Update
60 // The measurement value: z1 = 49.95°C. Since the measurement error is σ =
61 // 0.1, the variance σ^2 would be 0.01, thus the measurement uncertainty is:
62 // r1 = 0.01. The measurement error (standard deviation) is 0.1 degrees
63 // Celsius.
