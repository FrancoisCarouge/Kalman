139 // As you can see, the Kalman Filter tracks the vehicle quite well. However,
140 // when the vehicle starts the turning maneuver, the estimates are not so
141 // accurate. After a while, the Kalman Filter accuracy improves. While the
142 // vehicle travels along the straight line, the acceleration is constant and
143 // equal to zero. However, during the turn maneuver, the vehicle experiences
144 // acceleration due to the circular motion - the angular acceleration.
145 // Although the angular acceleration is constant, the angular acceleration
146 // projection on the x and y axes is not constant, therefore ax and ay are not
147 // constant. Our Kalman Filter is designed for a constant acceleration model.
148 // Nevertheless, it succeeds in tracking maneuvering vehicle due to a properly
