Detailed Description

Function object for providing an Eigen identity matrix.

Implemented with the Eigen linear algebra library matrices with sizes fixed at compile-time.

Note
Could this function object template be replaced by a variable template? Proposed in paper P2008R0 entitled "Enabling variable template template parameters".
