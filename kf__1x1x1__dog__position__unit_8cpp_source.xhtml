72 kalman filter{
73 // This is the dog's initial position expressed as a Gaussian. The state X
74 // position is 0 meters.
75 state{position{1. * m}},
76 // The measured output position Z.
77 output<position>,
78 // We are predicting that at each time step the dog moves forward one
79 // meter. This is the input U process model - the description of how we
80 // think the dog moves. How do I know the velocity? Magic? Consider it a
81 // prediction, or perhaps we have a secondary velocity sensor. Please
82 // accept this simplification for now.
83 input<position>,
84 // The variance to 400 m, which is a standard deviation of 20 meters. You
85 // can think of this as saying "I believe with 99.7% accuracy the position
86 // is 0 plus or minus 60 meters". This is because with Gaussians ~99.7% of
87 // values fall within of the mean. The estimate uncertainty P:
88 estimate_uncertainty{20. * m2},
89 // Variance in the dog's movement. The process variance is how much error
90 // there is in the process model. Dogs rarely do what we expect, and
91 // things like hills or the whiff of a squirrel will change his progress.
92 // The process uncertainty Q:
93 process_uncertainty{1. * m2},
94 // Variance in the sensor. The meaning of sensor variance is how much
95 // variance there is in each measurement. The output uncertainty R:
96 output_uncertainty{2. * m2}};
97
98 filter.predict(position{1. * m});
99 filter.update(position{1.354 * m});
100 filter.predict(position{2.352 * m});
101 filter.update(position{1.882 * m});
102 filter.predict(position{3.070 * m});
103 filter.update(position{4.341 * m});
104 filter.predict(position{4.736 * m});
105 filter.update(position{7.156 * m});

117 filter.update(position{14.8 * m});
