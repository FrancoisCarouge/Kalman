Eigen-based Kalman filter.

Implemented with the Eigen linear algebra library matrices with sizes fixed at compile-time.

Template Parameters
TypeThe type template parameter of the matrices data.
StateThe non-type template parameter size of the state vector x.
OutputThe non-type template parameter size of the measurement vector z.
InputThe non-type template parameter size of the control u.
PredictionArgumentsThe variadic type template parameter for additional prediction function parameters. Time, or a delta thereof, is often a prediction parameter.
Examples
building_height.cpp, dog_position.cpp, liquid_temperature.cpp, rocket_altitude.cpp, and vehicule_location.cpp.
